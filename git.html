Comandos de configuracion de Git

  • git config --global --list:Muestra correo y nombre de usuario
  • git config --global user.name fulanito:Sirve para crear mi nombre de usuario
  • git config --global user.name:Sirve para ver mi nombre de usuario
  • git config --global user.email mail@mail.com:Sirve para crear mi correo
  • git config --global user.email:Sirve para ver mi correo
