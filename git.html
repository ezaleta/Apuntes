Comandos de configuracion de Git

  • git config --global --list: Muestra correo y nombre de usuario
  • git config --global user.name fulanito: Sirve para crear mi nombre de usuario
  • git config --global user.name: Sirve para ver mi nombre de usuario
  • git config --global user.email mail@mail.com: Sirve para crear mi correo
  • git config --global user.email: Sirve para ver mi correo

    Flujo del trabajo de Git

  1. git init: Sirve para inicializar Git. Git registrara todos los cambios en el proyecto.
  2. git status: Muestra el estatus de los archivos
  3. git add NombreDelArchivo: Añade el archivo especificado al stage
  4. git add -A: Añade todos los archivos al stage
  5. git commit -m "texto": Carga los cambios a la rama actual
  6. git branch: Muestra todas las ramas existentes
  7. git branch NombreDeLaRama: Crea una rama
  8. git checkout NombreDeLaRama: Me mueve a la rama
  9. git merge NombreDeLaRama: Fusiona la rama al master
  10. git show NombreDeLaRama: Me muestra todas las modificaciones que se hicieron en esa rama
  11. git log: Me muestra todos los commit existentes
