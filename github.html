Configuracion de repositorio de github

  • Como crear una llave ssh.Sirve para conectar mi computadora con mi cuenta de github.

    Para ingresar la llave ssh en mi cuenta de github, debo darle click en mi profile y seleccionar la opcion settings, despues seleccionar la opcion ssh keys and gpgy me llevara a la opcion de crear una llave ssh nueva y ahi ingreso mi llave.

  • github pageEs un servicio de github que sirve para poder ver el frontend de mi proyecto. Si tengo archivos de lenguaje backend no los podre visualizar ya que solamente ejecutara archivos html, css y javascript.

    Para llegar a github page, necesito darle click en settings luego darle click en github page, y seleccionar en branch la rama master, y en unos minutos aparecera una liga que me dara github, el servicio es gratuito.

Comandos para trabajar con el repositorio

  • git remote add origin "url":Sirve para conectar los archivos locales al repositorio en github.
  • git remote -v:
  • git push origin master:Sirve para añadir la rama "master" o rama principal al repositorio.
  • git push origin NombreDeLaRama:Sirve para añadir la rama "NombreDeLaRama" al repositorio.
